# FinanzasPersonalesExcel
Este proyecto es un script de Google Apps Script diseñado para automatizar la gestión de cuotas de tarjetas de crédito (ej. VISA) y cálculos de ahorro en Google Sheets. Permite actualizar automáticamente los montos de cuotas, ajustar el sueldo disponible después de pagos y generar predicciones de próximos cobros.
